This supplementary material is a Matlab demo code to test the dual-pathway rectifier neural network for non-blind image deconvolution. 

The following files are included: 
--source code files: demo.m, columns2im.m, deblurPatch.m, image2cols.m, getPSNR.m
--trained models for kernel 1 and kernel 2: model_kernel1.mat and model_kernel2.mat
--test image: Cameraman.png


Running:
To run this program, open Matlab and run:
>> demo
